# Natural Language Processing (NLP) Task Categories — Complete Study Guide

This document provides a complete overview of Natural Language Processing (NLP) task categories.
It is designed as a study and interview reference.

---

## 1. What Are NLP Task Categories?
NLP task categories describe the type of operation performed on text:
Input text → NLP task → Output.

---

## 2. Core NLP Task Categories

### A. Text-to-Text Generation
**Definition:** Generate text from text.

**Examples:**
- Chatbots
- Translation
- Paraphrasing
- Question answering

---

### B. Text Classification
**Definition:** Assign labels to text.

**Examples:**
- Sentiment analysis
- Intent detection
- Spam filtering

---

### C. Summarization
**Definition:** Condense long text into a shorter form.

**Types:**
- Extractive
- Abstractive

---

### D. Sentencizer (Sentence Segmentation)
**Definition:** Split text into sentences.

---

## 3. Additional NLP Tasks

- Named Entity Recognition (NER)
- Tokenization
- Part-of-Speech Tagging
- Text Similarity
- Semantic Search
- Information Extraction
- Machine Translation

---

## 4. NLP Tasks vs Outputs

| Task | Output |
|-----|-------|
| Generation | Text |
| Classification | Label |
| Summarization | Text |
| Sentencizer | Sentences |

---

## 5. Interview Cheat Sheet
- Generation: Create text
- Classification: Label text
- Summarization: Compress text
- Sentencizer: Split text

---

## 6. Final Takeaway
NLP task categories define what problem is being solved, while models define how well it is solved.
